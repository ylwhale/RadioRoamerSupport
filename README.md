# RadioRoamer GitHub Pages files

This folder is ready to upload to a GitHub repository for a GitHub Pages site.

## Included pages
- `index.html` → marketing / landing page
- `support/index.html` → public support page for App Store Connect
- `privacy/index.html` → privacy policy page
- `styles.css` → shared styling
- `assets/app-logo-v2.png` → app icon
- `assets/screenshot-map.png` → app screenshot
- `assets/screenshot-player.png` → app screenshot

## Recommended GitHub Pages URLs
If the repository is named `radioroamer` and published as a project site under `jingyuhuang.github.io`, the pages will be:
- `https://jingyuhuang.github.io/radioroamer/`
- `https://jingyuhuang.github.io/radioroamer/support/`
- `https://jingyuhuang.github.io/radioroamer/privacy/`

## Upload steps
1. Create or open your GitHub repo.
2. Upload all files from this folder to the repo root.
3. In GitHub, open **Settings → Pages**.
4. Set the source to **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)` folder.
6. Save.

After GitHub Pages finishes publishing, use these in App Store Connect:
- **Marketing URL** → `/`
- **Support URL** → `/support/`
- **Privacy Policy URL** → `/privacy/`
